# PCB LAYOUT DESIGNER

РУССКИЙ

Простой и понятный апплет для проектирования разводки на печатной плате для ручной пайки и навесного монтажа.
Ввиду того, что я увлекаюсь радиоэлектроникой, то у меня возникла необходимость развести навесной монтаж на печатной плате с Aliexpress, но ничего даже близко похожего я не нашел, потому решил воплотить эту идею в жизнь. Весь код, графика и вся логика написаны Google Gemini по моим промтам.

ENGLISH

A simple and intuitive applet for designing PCB layouts for manual soldering and point-to-point wiring.
Since I'm interested in electronics, I needed to create a point-to-point wiring layout for a PCB from Aliexpress. I couldn't find anything even close, so I decided to bring this idea to life. All code, graphics, and logic were written in Google Gemini based on my own specifications.
